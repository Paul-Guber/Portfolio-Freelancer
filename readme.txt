# Верстка сайта Portfolio Freelancer была реализована в VS Code с помощью макета в figma,
были использованы: gulp, sass, js.

# В данной верстке:

- присутствует кнопка переключения темы (тёмная / светлая) 

- тема меняется в зависимости от системных настроек автоматически

- адаптивная верстка для мобильных устройств 320px.
